API синонимайзера, рерайтера, перефразировщика предложений - Sinoni.men

Наше API подойдет для больших объемов рерайта текста.

POST запросы отправлять на api.sinoni.men

Отправляете задание на синонимацию текста

Параметр: token - ваш токен, который вы получаете в панели управления;

Параметр: text - ваш текст, который требуется уникализировать.

При успешном принятии текста в задание, вы получите уникальный ключ {"result":{"id":"0246781ad2ae145e2841f0a740daf0d2"}}, по которому в следующем запросе получите результат синонимации вашего текста.

Получение результата рерайта текста

После успешной отправки текста в сервис, вы можете начать отправлять запросы на предмет того, готов ваш текст или нет.

Параметр: token - ваш токен, который вы получаете в панели управления;

Параметр: id - ID текста, который хотите получить.

Если ваш текст уже готов, вы получите ответ {"result":{"text":"Ваш уникальный текст"}}. Если же ваш текст еще проходит обработку, результат будет ошибка со статусом 181 {"error":{"code":"181"}}

Список возможных ошибок при работе с API

181 - Ваш текст еще не готов

183 - Ошибка сервера. Попробуйте позднее
